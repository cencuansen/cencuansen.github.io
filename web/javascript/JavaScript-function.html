箭头函数特点

  • 不能做构造函数，不能 new
  • 没有 arguments，可用剩余参数代替
  • 没有自己的 this，this 取决于箭头函数所在上下文
  • apply、bind、call 无法影响 this 指向
  • 没有原型属性
  • 不能作为 Generator 函数，不能使用 yield
