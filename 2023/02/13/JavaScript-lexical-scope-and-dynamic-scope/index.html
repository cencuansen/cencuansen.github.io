JavaScript 词法作用域和动态作用域

日期： 2023年 02月    分类： JavaScript
标签： # JavaScript # Function
← 返回

说明

JavaScript 中，变量采用的是词法作用域。

词法作用域/静态作用域

作用域在变量定义时就确定。

动态作用域

作用域在变量使用时才确定。
