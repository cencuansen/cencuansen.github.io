JavaScript this

Author: sanhuo Date: February 13, 2023  19:54:25

普通函数

普通函数中的 this 是根据运行时上下文动态确定的，this 指向函数运行时所在上下文。

箭头函数

箭头函数的中 this 在非运行期间就确定了，箭头函数的 this 和外层函数的 this 一致。
