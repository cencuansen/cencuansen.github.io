JavaScript apply、bind、call

Author: sanhuo Date: February 13, 2023  20:53:10

用途

apply、bind、call 用来改变函数内 this 指向

用法

区别

分类 说明
apply 参数数组形式传，并调用对应函数
bind 参数一个一个传，返回一个函数
call 参数一个一个传，并调用对应函数

问题

问：一个方法连续 bind 多次，结果如何？
答：只有第一次 bind 有效果。
