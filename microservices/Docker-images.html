Docker 镜像和容器

日期： 2023年 02月    分类： Docker
标签： # Docker
← 返回

镜像

镜像是一种轻量级、可执行的独立软件包，用来打包软件运行环境和基于运行环境开发的软件，它包含运行某个软件所需的所有内容，包括代码、运行时库、环境变量和配置文件。
镜像由文件系统叠加而成，最底端是一个引导文件系统(bootfs)。

容器

当一个容器启动后，镜像会被移到内存中，而引导文件系统则会被卸载。
